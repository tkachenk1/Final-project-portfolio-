Text Input Field with Label

CSS selector:

.input-label for label and .input-field for the input

Sample code:

<label class="input-label">
 Label here
 <input type="text" name="demo-text" class="input-field">
</label>

Rendered element:

Radio Button with Label

CSS selector:

.input-label for label and .input-radio for the input

Sample code:

<label for="demo-radio-input" class="input-label">Label here</label>
<input id="demo-radio-input" type="radio" name="demo-radio" class="input-field">

</a>
